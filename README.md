### 简介
```
屆時遊戲成品將會有4條燈帶來模擬不同的訊息傳輸速度：2G、3G、4G、5G，目前Demo先以一條燈帶和控制杆來進行演示，在影片中，可以觀看到在按下按鈕後，光在燈帶中的傳輸狀況。

控制杆的四個方向分別代表由快至慢的4個傳輸速度，通過比較傳輸速度的不同和LED燈帶的變化，顯示不同傳輸速率對訊息傳輸的影響。
```
